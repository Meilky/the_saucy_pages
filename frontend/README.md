# The Saucy Pages Frontend

This is the frontend of The saucy pages.

## Requirements

You need to install [nodejs](https://nodejs.org/en).

## Setup

To install dependencies:

```bash
npm install
```


## Starting

```bash
npm run dev
```

By default, it runs on `http://0.0.0.0:5055`.

## Building

```bash
npm run build
```
